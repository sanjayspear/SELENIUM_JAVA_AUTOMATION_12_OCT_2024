git init: Initialize a new Git repository.
git clone: Clone an existing repository into a new directory.
git add: Add files to the staging area.
git commit: Commit the staged files with a message.
git status: Show the status of the working directory and staged changes.
git remote: Manage set repositories tracked by your local repository.
git push: Push the changes from your local repository to the remote repository.
git branch: List, create, or delete branches.
git log: View the commit history.
git diff: Show the changes between commits, branches, or files.
git reset: Unstage or revert changes in your working directory.
git rm: Remove files from the working directory and staging area.
git remote -v: Displays the URLs that are set as remote repositories for your local project.
git remote remove origin: Removes the remote repository named origin from your local repository
